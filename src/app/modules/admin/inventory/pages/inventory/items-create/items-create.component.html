Información

Complete el formulario para crear un nuevo elemento en el sistema de inventario.

Los elementos de inventario representan los productos o materiales almacenados en los diferentes almacenes.

Asegúrese de proporcionar información precisa, especialmente para los campos obligatorios marcados con asterisco (*).

Consejos

Stock mínimo: Establece el nivel mínimo de stock que debe mantenerse para este elemento.

Stock máximo: Establece el nivel máximo de stock que debe mantenerse para este elemento.

Punto de reorden: Nivel de stock en el que se debe realizar un nuevo pedido.

Ubicación: Especifique la ubicación exacta dentro del almacén para facilitar la localización.
